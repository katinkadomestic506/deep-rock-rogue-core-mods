4. Find your Deep Rock Galactic: Rogue Core installation folder. This is usually located inside your Steam installation directory under steamapps/common/Deep Rock Galactic.
5. Open the folder named `Content` and find the `Paks` folder.
6. Copy the mod files you extracted from the .zip package.
7. Paste these files into the `Paks` folder.
8. If the computer asks to replace existing files, choose skip.

## 💡 Frequently Asked Questions

Read these answers if you run into trouble during setup.

Does this change my game save?
These mods change visual elements and game logic. They do not delete your progress. Always keep a backup of your save file folder.

Can I play online with these mods?
Some mods work in multiplayer sessions while others trigger game mismatches. Start a private game if you experience connectivity issues while using custom files.

How do I remove the mods?
Return to the `Paks` folder inside your game directory. Delete the mod files you added earlier. Start the game again to restore your original settings.

What if the game crashes?
Remove the mod files from your `Paks` folder. Verify your game files through the Steam library. Right-click the game in your library, select Properties, go to Installed Files, and click Verify integrity of game files.
